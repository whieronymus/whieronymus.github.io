Welcome to BBC v2 — Season 2025–2026 Request Changes

This is a high-tech fantasy football league built for competitive friends, clean ops, and top-tier banter.

We’ll be requesting league input from time to time on rules, settings, and new ideas. Participation is expected — your voice helps shape the league.

So strap in, hold onto your boots, and get ready… because this season is going to blow your socks off.

Waivers & Transactions Request Changes

  • Waiver Type: FAAB (Free Agent Acquisition Budget). Request Changes
  • Budget: $200 (virtual). Bids are blind; highest bid wins. $0 bids allowed. Request Changes
  • Processing: Primary run: Wednesday 10:00 AM. Secondary runs daily at 10:00 AM for dropped players. Request Changes
  • Add/Drop: Continuous waivers during the week; post-waiver players become free agents until game lock. Request Changes
  • Trades: Deadline at NFL Week 11 end. Review by commissioner; veto for collusion only. Request Changes
  • IR: Disabled for the 2025–2026 season. Request Changes

Playoffs Request Changes

  • Teams: 6 of 12 qualify (Seeds 1–6) Request Changes
  • Byes: Top 2 seeds receive first-round byes Request Changes
  • Weeks: NFL Weeks 15–17 Request Changes
  • Seeding: Record → Points For → Head-to-Head Request Changes
  • Toilet Bowl: Optional consolation bracket for non-playoff teams Request Changes

League Policies Request Changes

  • Sportsmanship: Trash talk encouraged; hate speech/personal attacks prohibited. Request Changes
  • Lineup Deadlines: Player spots lock at individual game kickoff. Request Changes
  • Inactive Managers: Commish may set a fair lineup if two consecutive weeks are neglected. Request Changes
  • Rule Changes: Off-season voting, simple majority unless stated. In-season changes require 2/3 and cannot affect prior results. Request Changes
  • Money: Any buy-ins/payouts handled off-platform between members. Request Changes
  • Ties: Regular season ties stand. Playoff ties: bench points, then higher seed. Request Changes

Optional Variants (Commissioner Picks) Request Changes

Replace one FLEX with a Superflex (QB/RB/WR/TE) to increase QB value and draft diversity. Request Changes

Waiver Order Variants

  • Continual Rolling List: After a successful claim, that manager moves to the end of the list. Request Changes
